# ⛪ Gestão de Visitantes - Igreja

Sistema desenvolvido para facilitar o cadastro de visitantes e seus familiares, além de fornecer um mural de avisos dinâmico e relatórios em Excel para a secretaria da igreja.

## 🚀 Status do Projeto
> 🟢 **Sprint 1 Concluída:** Configuração de ambiente, Banco de Dados (PostgreSQL) e Entidades JPA.

## 🛠️ Tecnologias Utilizadas
* **Java 21**
* **Spring Boot 3.4.3**
* **Spring Data JPA** (Persistência de dados)
* **PostgreSQL** (Banco de dados relacional)
* **Lombok** (Produtividade no código)
* **Bootstrap 5** (Frontend responsivo - *em desenvolvimento*)
